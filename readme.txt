== Description ==
Live Talk system provides website owners with a comprehensive set of tools for customer care that no tools have yet. 
OMI Live Talk is available for free for business owners for a lifetime, now

== Installation ==
1. From your WordPress Dashboard go to \'Plugins\' > \'Add new\' and search for \'OMI Live Talk\'.
2. Click \'Install Now\' under the title of the OMI Live Talk plugin
3. Click activate
4. Go to \'Menu Dashboard\' > \'OMI Live Talk\' and check the box to activate the domain name your OMI Live Talk System.
5. Click \'Save\' and you\'re done!

Or:

1. Upload the `omi-live-talk`-folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to \'Settings\' > \'OMI Live Talk\' and check the box to activate the domain name your OMI Live Talk System.
4. Click \'Save Change\' and OMI Live Talk is live!
